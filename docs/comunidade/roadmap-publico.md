# Roadmap Publico (2026)

## Fase 1 - Base aberta (concluida)

- Site Docusaurus com trilha completa SSA.
- Labs, templates, certificacao e carreira.
- Publicacao em GitHub Pages.

## Fase 2 - Comunidade e colaboracao (em andamento)

- Guia de contribuicao e governanca.
- Templates de issue e PR.
- Backlog publico priorizado.

## Fase 3 - Especializacoes por dominio (proxima)

- Trilha SSA para Saude.
- Trilha SSA para Juridico.
- Trilha SSA para Suporte B2B.

## Fase 4 - Plataforma de pratica (proxima)

- Banco de casos para evals comunitarias.
- Colecao de capstones referencia.
- Matriz de maturidade SSA por organizacao.

## Backlog priorizado

1. Guia de arquitetura para RAG em producao.
2. Modulo de custos e otimização economica em sistemas de IA.
3. Pacote de exemplos adversariais para seguranca.
4. Trilha de onboarding corporativo SSA.
5. Traducao completa para ingles.
